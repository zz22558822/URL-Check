# 網頁有效檢查器
## URL-Check

![URL Check cover](https://github.com/zz22558822/URL-Check/blob/main/img/URL-Check.png)

---

## 使用方法:
1. 將要檢查的網址填入 URL.txt (多網址請換行)  
2. 運行 URL-Check.exe


## 檔案結構
File/ → 驅動項目  
├── Log/  
├── URL-Check.exe  
└── URL.txt  

Log/ → 統計資訊 (檔名包含時間)  
URL-Check.exe → 主程序  
URL.txt → 放要檢查的網址
